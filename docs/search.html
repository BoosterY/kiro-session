🔍 Search

Find any conversation instantly. Two modes: fast for speed, smart for semantics.

Two Search Modes

⚡

fast mode (default)

FTS5 full-text search. <50ms. Prefix matching, hyphen normalization, BM25 ranking, context snippets. Zero tokens.

🧠

smart mode (--smart)

LLM judges relevance from session summaries. Cross-language matching. Returns a one-sentence explanation per result.
