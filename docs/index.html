kiro-session

Smart session management for Kiro CLI. Find, search, split, and organize your AI conversations.

v0.3.0 · 19 KB · Zero Config

The Problem vs The Solution

😤 Before

  • Session list shows only IDs and one-line summaries
  • Can't search across session content
  • 200-turn sessions mixing 5 different topics
  • No way to find "that Docker conversation from last week"
  • Stale sessions pile up with no cleanup
  • Resuming a specific session = scrolling through IDs

🚀 After

  • Rich interactive browser with names, topics, and metadata
  • Full-text search across all conversation content
  • LLM splits long sessions into focused topic-based ones
  • Find any session by keyword, directory, or recency
  • Smart cleanup suggestions with safety guardrails
  • One-command resume with precise session targeting

Key Features

🔍

Smart Search

Search by keyword, directory, or recency. Deep search scans full conversation content across all sessions.

🧠

LLM-Powered Index

Generates meaningful session names and topic summaries. Runs in background — zero effort, richer info.

✂️

Semantic Topic Splitting

LLM groups turns by meaning, not sequence. Review, adjust with natural language, and your preferences are learned.

⏪

One-Command Resume

Select any session and get a precise resume command. Works with both terminal and TUI mode.

🧹

Safe Cleanup

Smart suggestions for stale sessions. No auto-deletion — you always confirm. Split parents reclaimable after 30 days.

⚡

Zero Config

Works alongside native Kiro CLI. No API keys needed — uses kiro-cli's own LLM. Index self-heals on external changes.
